# Data Visualization

## Description

This project visualizes the cleaned book dataset created in Task 2 using Matplotlib.

## Features

- Bar chart of book ratings
- Histogram of book prices
- Pie chart of book availability
- Automatically saves charts as PNG images

## Technologies Used

- Python
- Pandas
- Matplotlib

## Files

- visualization.py
- charts/
- README.md
- requirements.txt

## How to Run

```bash
pip install -r requirements.txt
python visualization.py
```

## Output

The following charts are generated inside the `charts` folder:

- ratings_bar_chart.png
- price_histogram.png
- availability_pie_chart.png
