# Exploratory Data Analysis (EDA)

## Description

This project performs exploratory data analysis on the book dataset created in Task 1.

## Features

- Load CSV dataset
- Explore dataset structure
- Clean price column
- Convert price to numeric values
- Generate statistical summary
- Find highest and lowest priced books
- Count ratings
- Count availability
- Save cleaned dataset

## Technologies Used

- Python
- Pandas

## Files

- analysis.py
- cleaned_books.csv
- README.md
- requirements.txt

## How to Run

```bash
pip install -r requirements.txt
python analysis.py
```

## Output

A cleaned dataset named `cleaned_books.csv` is generated.
